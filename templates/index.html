Una idea en desarrollo

Ejemplos de invocacion a la API:


Send text message

curl -v -G http://localhost:8080/send -d token=1234asdf --data-urlencode 'message=El proceso se completó exitosamente.' -d severity=0

Send image with caption

curl -X POST -F 'image=@./telegrams.png' -F token=1234asdf http://localhost:8080/post -F severity=1 -F message="El proceso se completó con errores."
